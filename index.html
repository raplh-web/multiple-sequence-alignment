Multiple Sequence Alignment using Ant colony Optimisation

Multiple Sequence Alignment (MSA) is a process through which study of closely related genes becomes easier. The problem is to find an optimal set of sequences between multiple sequences of DNA, RNA or Proteins. It is very helpful to understand the evolutionary relationships between different genes[1]. This project implements a nature inspired metaheuristic- Ant Colony Optimization to solve the MSA problem in high-performance Julia language. We will be working on converting the MSA problem to Travelling Salesman Problem (TSP). TSP is a classic example of NP-Hard combinatorial problems in Computer Science. A lot of optimization algorithms are available to solve the TSP problem. Ant Colony Optimization (ACO) is one of the nature inspired metaheuristic which solves the TSP problem. ACO is a population based metaheuristic and can be used as approximation solutions to combinatorial problems [2]. In the evaluation phase, we will be comparing the results to the existing implementations of the ACO algorithms. The metrics for the evaluation phase will be time taken to do the same approximation, memory utilization.

From Algorithms to Code

The coding approach for this problem will contain two phases. One being the Conversion of the Multiple Sequence Alignment to Travelling Salesman Problem. And the other phase of computationally Solving the Travelling Salesman Problem using Ant - Colony Optimization.

The main funcitons of the computational solution will be:

  • MSA_to_TSP(sequences): To convert the MSA problem to a travelling salesman problem. The sequences is an array which stores the sequences to be aligned.
  • get_alignment_score(v,w): To get the alignment score between two sequences where v and w are the input sequences.
  • create_colony(num_ants, num_nodes): Creating an array colony storing dictionory of 2 key value pairs: path and distance, with size = number of ants. (num_ants = total no. of ants, num_nodes = total no. of nodes)
  • create_pheror_matrix(num_nodes): To create a pheromone matrix. (num_nodes = total no. of nodes)
  • calculate_proba(num_nodes, pheromone, distance_matrix, alpha, beta): For creating a probablity matrix of size N x N where N = num_nodes. (alpha and beta assigned constants)
  • calculate_proba_ant(pheromone, distance_matrix, unvisited_nodes, current_node, proba, alpha, beta): To calculate probablity to one node to the current node.
  • find_best_path(n_ants, colony): To find the best paths.
  • update_pheror_matrix(num_nodes, n_ants, pheromone, distance_matrix, colony, Q, decay): To update the pheromone matrix after each ant's traversal
  • calculateDist_ant(ant, colony, distmatrix): To calculate the total distance of each ant's path
  • traverse(ant, num_nodes, colony, pheromone, distance_matrix, proba, alpha, beta): In order to simulate the traversal of ant (the variable 'ant' is the index of ant)
  • run1(num_ants, num_nodes, distance_matrix, iterations, Q, decay, alpha, beta): Main code which integrates the rest of functions inorder to execute the complete ACO.
  • find_gap_indices(A, alignedA): Find the gap(-) index
  • insert_gaps(S,gap_indices_for_A): Insert gaps in the sequence at desired indecies
  • align_output_sequences(sequences, order): To align the otput sequence
